## Project Description
Wordly is a Single Page Application (SPA) that provides an interactive dictionary experience. Users can search for English words and instantly view their definitions, pronunciation, example usage, synonyms, and pronunciation audio without refreshing the page.

This project demonstrates modern front-end web development practices using **HTML, CSS, and JavaScript**, including API integration, event handling, and dynamic DOM manipulation.

## ğŸš€ Features
- Search for English words using a public dictionary API
- Display word definitions, part of speech, and example usage
- Play pronunciation audio when available
- Display synonyms when provided by the API
- Error handling for invalid or empty searches
- Responsive and user-friendly interface
- Single Page Application behavior (no page reloads)
